パイプボルトアクション

DATE: 2025/11/19

Fallout4、Fallout76

パイプボルトアクションは、モジュール式武器です。

特徴

パイプボルトアクション・ピストルは、木製の本体と、スクラップメタル、スプリング、ボルト、パイプを組み合わせて作られた作動部品、そしてグリップで構成される即席の武器です。

  • 弾薬: .308弾を発射し、銃の左側から装填される内蔵マガジンを使用します。

  • 変形性: ストックを改造すると、パイプボルトアクション・ライフルになります。

  • 耐久性: パイプボルトアクションは役立つ武器になりえますが、機械加工された銃に比べて状態の劣化が速いという欠点があります。
    この速い劣化は、Licensed Plumber Perkを振ることで相殺されます。

 入手場所

武器作業台で最初からクラフト可能です。
スコーチが使用しています。

性能データ

  • 種類 : 遠距離武器（ピストル / ライフル）

  • 作成可能レベル : 1 / 5 / 10 / 20 / 30 / 40 / 50

  • ダメージ : 99 / 111 / 125 / 159 / 203 / 259 / 330

  • Ammo : .308弾（ウルトラサイト弾も使用可能）.38弾（改造により使用可能）

  • 装弾数 : 61

  • Fire rate : 31

  • Range : 137

  • Accuracy : 59

  • AP : 18

  • Weight : 3.0

感想

この武器は、序盤からクラフト可能で、改造によって比較的高い単発ダメージを出せるのが魅力ですが、連射速度は遅く、耐久性が低いという特徴があります。
パイプ系列は即席の武器で◯◯体倒す系のChallengeに使われるのと、柔軟な改造な特徴ですね。
私のロア用のキャラは使ってますが、リロードがながぁいのがボルトアクション系の弱点ですね。
ちゃんとオンスロートとかリロード速度とかを積むと良いのだと思いますが、初心者の時は特に使いづらいと思う武器だと個人的には思います。
