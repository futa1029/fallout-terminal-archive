ゼータ星人

DATE: 2025/11/16

ゼータ星人 は、グレート・ウォーの前と後の両方で地球とその住人の出来事に関与してきた異星人の種族です。彼らは『Fallout』シリーズに複数回登場しています。

背景

起源

ゼータ星人が彼らの惑星から太陽系へ旅をする動機については、ほとんど何も知られていません。
彼らの目的は、人間やクリーチャーの誘拐、研究のための世界中のオブジェクトの収集、極低温保存、および実験に対する関心だけです。

サンプルを誘拐

彼らは地球に到着して以来、千年間にわたり、地球から人間や他のクリーチャー、そして地球の技術のサンプルを誘拐してきました。
彼らが地球に存在していた証拠は、日本の江戸時代（16世紀後半から17世紀初頭）にまで遡ります。

彼らの影響が現代社会で感じられるようになったのは、1960年代にHartigan大佐とClarabella 7カプセルがアメリカ合衆国から誘拐された後のことでした。
Quaere Verumのようなグループが彼らに関する真実を明らかにしようと結成されましたが、このグループは一般には「奇妙な陰謀論者」と広く見なされていました。

戦前の活動

  • アパラチア :
    戦前のアパラチアでは、ゼータ星人はフラットウッズの町の周辺の荒野で個人を捕獲し、実験を行うゼータ星人の工作員ネットワークを通じて活動していました。

  • Vault-Tec CEO :
    そのうちの一人は、2064年から2069年頃にVault 76の建設を視察していたVault-Tecの副最高経営責任者でした。

フラットウッズ・モンスター

  • フラットウッズ・モンスター :
    これらのエイリアン工作員はフラットウッズの地元住民によって頻繁に目撃され、報告されましたが、地元の法執行機関はこれらの目撃情報を真剣に受け止めませんでした。
    エイリアン工作員は地元住民やメディアによって「フラットウッズ・モンスター」と呼ばれました。

  • 高官の誘拐 :
    核戦争の直前まで、ゼータ星人は戦前のアメリカ合衆国全土で人間の誘拐を続けました。
    既知の誘拐被害者には、言語学の専門家、アメリカ合衆国上院議員、そして当時のアメリカの核ミサイル（ICBM）を全て発射するためのコードを持っていたホワイトハウス内部の人物が含まれていました。

  • アンカレッジ奪還作戦 : 
    アラスカのアンカレッジ奪還作戦（資源戦争の一部）中、2066年から2077年の間に、数多くのアメリカ陸軍の兵士がゼータ星人によってアラスカから誘拐されました。

戦後

  • 変異体の研究 : 
    核戦争後も、ゼータ星人は放射能汚染されたウェイストランドから生じた新しいクリーチャーを誘拐、研究、実験し続けました。
    特にキャピタル・ウェイストランドでは、スーパーミュータント、フェラル・グール、バラモンなどを捕獲しました。

エンクレイヴの研究

  • エンクレイヴの研究 :
    戦後何年もの間、エンクレイヴはアパラチアでゼータ星人の遺骸を研究するための研究施設を維持していました。

Fallout 76

  • 『Fallout 76』での侵攻 :
    2104年頃、ゼータ星人の艦隊が周期的にアパラチアに侵攻し始め、Vault居住者やホーマー・サパースタインという謎の隠遁者を含む、この地域に住む他のグループによって阻止されました。

  • 『Fallout 3』での遭遇 :
    2277年頃、Lone Wandererが信号に導かれ、キャピタル・ウェイストランドに墜落したゼータ星人の偵察機「Theta」に遭遇します。
    その後、ゼータ星人のマザーシップ「Zeta」に連れ去られ、実験台にされますが、他の捕虜たちと共に脱出します。

社会

ゼータ星人の社会について知られていることは、趣味や個人的な追求に関心がなく、仕事と進歩に文化的な焦点を当てていることを示唆しています。
彼らの建築は簡素で純粋に機能的であり、居住空間には個性化の要素が欠如しています。
彼らは静止カプセルと呼ばれる無菌の金属製カプセルで眠っているようです。
マザーシップには娯楽の手段は見当たりません。

しかし、エイリアンの労働者が攻撃されたときに明らかな恐怖を示すことから、ある程度の個性や感情を持っていることが示唆されます。

マザーシップ「Zeta」の乗組員は、ギディアップ・バターカップのおもちゃに対して集団的な魅了を共有しています。
捕虜の一人であるサリーは、自分がそれに注意を向けたときにエイリアンが非常に興奮したことに気づいていますが、なぜ彼らがそう行動するのかは知りません。

言語

ゼータ星人は、甲高い金切り声や泣き声で構成される独自の言語を話しているようです。
このため、人間とエイリアンとの間の言葉によるコミュニケーションは今のところ不可能です。
あるいは、エイリアンが単に議論に興味がない可能性もあります。

軍事

サポート・ドローン

  • 戦闘方法 :
    マザーシップ「Zeta」のゼータ星人は、その虚弱な体のため、主にガーディアン・ドローン、サポート・ドローン、自動タレットに戦闘の大部分を依存しています。

  • 兵士 :
    特定のゼータ星人は戦闘準備の整った兵士のようで、防御のためにパーソナル・フォースフィールドとフォースフィールド・ジェネレーター、遠距離戦闘のためにアトマイザーとディスインテグレーター、近接戦闘のためにショック・バトンを使用します。
    彼らの武器はすべてエネルギーベースです。

  • 技術支援 :
    彼らはヒーリング・ジェル、高度なヒーリング・アーチウェイ、テレポート・マトリックスなどの支援技術にも頼っています。

制作裏話

Fallout 3デザイン

  • デザイン :
    ゼータ星人は、『Fallout 3』のためにAdam Adamowiczによってデザインされました。
    プロテクトロンと同様に、そのデザインは20世紀の「火星人」を連想させるパルプ・サイエンス・フィクション映画を強く想起させます。

  • アルミホイルスーツ :
    彼らの環境スーツの見た目がアルミホイルに似ているのは、ロズウェル事件の伝説に関連した、政府が農民から押収したとされる金属がアルミホイルに似ていたという話からインスピレーションを受けた可能性があります。

Fallout 4以降

  • 『Fallout 4』のデザイン :
    『Fallout 4』で改訂されたゼータ星人のデザインは、1996年の映画『マーズ・アタック!』に登場する火星兵のユニフォームに酷似しています。

  • 「Zetan」の名前の由来 :
    「Zetan」という名前は、ポップカルチャーで一般的に使われるステレオタイプのエイリアンである「Zeta Reticulans（ゼータ・レティキュリアン）」に基づいています。

  • Men in Black :
    「Men in Black（黒服の男たち）」の伝説は、ロズウェル事件の余波で政府のエージェントが民間人を脅迫したという未確認の報告に由来する可能性があります。

  • 『Mothman Prophecies』との関連 :
    「黒服の男たち」が変装したエイリアンであるという概念は、1975年の書籍『モスマンの予言』によって広まりました。
    『Fallout 76』の（使者）は、黒服の男の描写に合致するキャラクターです。
