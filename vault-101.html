Vault 101

DATE: 2026/2/3

Vault 101 は、2277年時点でキャピタル・ウェイストランドに位置する Vault-Tec 製の Vault です。
バージニア州のワシントンD.C.の北西、メガトンの近く、スプリングベール の西にあります。
ここは、孤独な放浪者 (Lone Wanderer) が父親を探しに出発するまでの住処です。

背景

建設と目的

この Vault は2060年代に、ワシントンD.C.の他の Vault と共に建設されました。
Vault 76 の監督官が当初、最初の監督官として選ばれていましたが、彼女はアパラチアに留まるためにその職を辞退しました。
Vault 101 の実験の本来の、そして潜在的な目的は、Vault が一度も開かなかった場合の監督官の役割を研究するために、無期限に閉鎖され続けることでした。
Vault 101 には、浄水器の予備部品など、無期限に機能するために必要な装備のほとんどが供給されていましたが、地上の居住者にしか役立たない装置である G.E.C.K. は支給されませんでした。
しかし、Vault 101 は本来の義務に違反して何度も開放されており、それらの出来事の最中にラッドローチが侵入しました。
時が経つにつれ、Vault はさらなる大規模な大量発生の影響を受けやすくなっています。
Vault 101 セキュリティの主な目的の一つは、これらの害虫の封じ込めとなっています。

初期の年月

2077年の最終戦争の直後、最初の Vault 101 監督官は、誰も Vault から出られないようにするために、外の世界は人が住めない核の荒野であり、Vault の外には生存の希望がまったくないという幻想を作り上げました。
これは、戦争前に録音された偽の定期的なラジオ放送によって達成され、Vault 101 の居住者たちの間で、生存の唯一の希望は Vault 内にあるという考えを強めました。
最初の監督官は2127年に自然死しましたが、後継者が Vault 実験を継続できるよう準備を整えていました。

二代目の監督官も同じ孤立主義の教義で居住者を導きましたが、年長の居住者たちを Vault 実験に加担させるよう説得も試みました。
居住者たちは「Vault で生まれ、Vault で生き、Vault で死ぬ」という Vault 101 のマントラに従って封印されていました。

2241年より前のいつか、ある無名の監督官は外部との接触を強く支持し、アン・パーマーが率いる調査チームを編成しました。
彼らは共に旅の計画を立て、遭遇する可能性のある変異のタイプを理論化し始めました。
彼らは最終的に 2241年2月10日より前に、メガトンまで足を運び、Vault 101 周辺の外の世界へと出向きました。
調査チームは2月10日に帰還し、採取したサンプルを分析して、外の世界がまだ危険な放射能を帯びているか、あるいは生命を維持できるかどうかを確認しました。
彼らは仕留めたジャイアント・アントを研究し、学んだ内容に照らして調査計画を修正することに決めました。
調査チームのメンバーであるルイスとアグネス・テイラー の二人は、二つの定住地間の連絡役を務め、周囲の環境をさらに調査するためにメガトンに残りました。
アグネス・テイラーイ�2258年頃に Vault 101 に帰還しました。
アルフォンス・アルモドバル の前任の監督官は、別のキャピタル・ウェイストランドへの遠征が失敗した後、姿を消しました。

監督官アルフォンス・アルモドバル

アルフォンス・アルモドバルは2258年より前に新しい監督官となりました。
彼の統治下で、Vault は厳格な孤立主義政策を掲げる警察国家へと変貌しました。
アルフォンスの任期の初期である2258年から2259年頃、ジェームス とその子供（後に孤独な放浪者として知られる）が避難所を求めて Vault 101 の前に到着しました。
ジェームスは入居の代償として Vault 101 の新しい医師になることを提案し、アルフォンスはその要求を聞き入れました。ジェームスはアルフォンスとの合意があるため、子供にその出自を隠し続け、孤独な放浪者には Vault 内で生まれたと信じ込ませました。

2260年代後半、ジョナス・パーマー が理性の声となることを期待され、アルフォンスによってジェームスの助手に任命されました。
2265年から2267年の間に、一人の少女が Vault 101 を脱出し、メガトンへ辿り着きました。
彼女は「クレーターサイド・サプライ」に入り、モイラ・ブラウンに Vault ジャンプスーツの補強を依頼しましたが、少女は二度とそれを取りに戻ることはなく、アーマード Vault 101 ジャンプスーツを後に残しました。

2268年7月12日、浄水器が故障し始めましたが、スタンリー・アームストロングによって修理されました。
翌日の7月13日、ブッチ・デロリア、ウォーリー・マック、ポール・ハノン・ジュニアは、後に「トンネル・スネーク」として知られることになる将来のギャングの名前を探し始めました。
2272年には別の居住者が脱出し、同じくメガトンへ辿り着きました。
彼は「モリアーティのサロン」に行き、コリン・モリアーティに対し、Vault 101 のメンバーは監督官こそが唯一の理性の声であると洗脳されていると語りました。

ジェームス（父）の出発

2277年8月17日、ラッドローチの大量発生が始まり、ジェームスはその出来事を有利に利用して素早く Vault を脱出しました。
続くラッドローチとセキュリティチームの衝突の中で、アグネス・テイラーは心臓麻痺で亡くなり、フロイド・ルイスとヴィッキ・ハノンは殺害され、ポール・ハノン・ジュニアは重傷を負いました。
ジョナス・パーマーは、なぜジェームスが去ったのかについて即座に尋問を受け、その後にスティービー・マックによって殺害されました。
セキュリティ部隊の全メンバーに対し、ラッドローチの発生を食い止めることと、ジェームスの出発について尋問するために孤独な放浪者を拘束することの両方が命じられました。
数分後、監督官アルモドバルの娘であり、孤独な放浪者の親友であるアマタ・アルモドバルが彼を起こし、状況を説明しました。

孤独な放浪者の脱出中、トム・ホールデンとメアリー・ホールデンは、警備員たちが害虫を殺そうとしている間に自分たちも去ることを検討します。孤独な放浪者は、パニックに陥ったブッチ・デロリアに声をかけられます。彼は、自室でラッドローチに襲われている母親を助けるために必死に助力を求めていました。その間、アマタは父親とマック隊員によって尋問を受けていました。

入り口まで突き進むと、アマタはパーク隊員とウルフ隊員 が到着する直前に孤独な放浪者に別れを告げます。
Vault の扉が開き、孤独な放浪者は Vault 101 を去り、記憶にある中で初めてキャピタル・ウェイストランドへと足を踏み入れました。

ジェームスの出発からしばらくして、ポール・ハノン・ジュニアは治療を拒否した後にラッドローチの噛み傷が原因で死亡し、監督官は以前よりも居住者に対して抑圧的になりました。
この経験を通じて、生き残った居住者たちは地上での生活が可能であることを知り、この発見が Vault 101 内に新たな危機と分裂を生み出しました。

Vault 実験の危機

アマタ・アルモドバルをリーダーとする反乱グループが台頭しました。
これには Vault 内の若い居住者全員が含まれており、彼らは Vault のクリニックに立てこもりました。
彼らは Vault 101 を開き、外の世界と交流し、近隣の定住地と限定的な取引を行い、必要に応じてウェイストランドへの遠征を行うことを望んでいました。
もう一方のグループは監督官に率いられ、Vault 実験に従って Vault を永久に閉鎖し続けることを望んでいました。
ブッチ・デロリアは Vault を完全に放棄し、地上で暮らすことを望んでいました。

その直後、Vault はエンクレイヴから、内部への立ち入りを求める無線連絡を受けました。
彼らは以前、現在は機能していないパスワードを介してアクセス権を持っていたと主張しました。
監督官は、孤立を維持したいグループの一員であったため、彼らの立ち入りを拒否しました。
彼は Vault とその居住者を未知の要因にさらしたり、自分たちの取り組みを変えようとする動きを警戒しました。
監督官は、反乱分子が Vault を開くのを防ぐという目前の極めて重要な任務から注意を逸らすことができませんでした。
それに応じる形で、アマタは緊急周波数を通じてキャピタル・ウェイストランドにメッセージを送り始め、孤独な放浪者が自分たちの闘いを助けに戻ってくることを期待しました。

余暇活動

Vault 101 の住民は、自由時間に様々な活動に従事しています：
「ケーキ焼き大会 」は競争イベントであり、出場者は様々な種類のケーキを焼き、選考基準に従って審査されます。
勝者にはふさわしい賞品が与えられます。
2268年7月13日、食堂内の掲示板にある通知で、恒例のケーキ焼き大会が告知されました。
今年の審査員はクグロフ型ケーキのみを受け付けます。

「創設者の日 (Founder's Day)」は Vault 101 のカレンダーで認められた祝日であり、10月23日に祝われます。
この休日は、21世紀後半に Vault を統治した最初の監督官を記念するものです。
2世紀後、居住者のベアトリス・アームストロングは、自作の古い詩の一つを彼に捧げることで、初代監督官を称え続けています。
直近の「創設者の日」には、ベアトリスは「監督官 」という言葉と韻を踏む言葉を見つけるのに苦労したため、新しい詩を書くことができませんでした。

「ミュータント狩り」は、Vault 101 で子供たちが遊ぶゲームです。
これはかくれんぼのポスト・アポカリプス版のバリエーションです。
一人の子供がミュータント役に選ばれて隠れ、他の子供たちが彼らを探します。
「ミュータント」が見つかるとゲームが再開され、別の子供がミュータント役に選ばれます。
2260年代のウォーリー・マックの誕生日パーティーの一つで、アトリウムにてこのゲームが遊ばれました。
ウォーリーは後に、2268年7月13日の孤独な放浪者の10歳の誕生日パーティーでこのゲームに言及しました。
このゲームを引き合いに出すことで、ウォーリーは自分のパーティーを放浪者のパーティーと比較して卑下していました。

レイアウト

「Vault 101 の住民は、時折発生するラッドローチの襲撃を除けば、外の世界から保護されたままである」

―― Fallout 3 ローディング画面

Vault 101 は、メインエントランス、監督官のトンネル、アトリウム/上層階、下層階の4つのセクションに分かれています。
各セクションはドアによって仕切られており、階段が二つのセクション内のフロアとレベルを繋いでいます。
レベルと部屋は標識によって識別されます。
キャピタル・ウェイストランドへの経路は二つあり、アトリウムからの直接の経路と、監督官のトンネルという隠されたトンネルがあります。
各セクションは垂直に積み重なっているわけではなく、Vault が地中深く掘り進められるにつれて、各セクションが徐々に西へと移動するようにずらして配置されています。

メインエントランス

このセクションは、キャピタル・ウェイストランドと Vault を繋ぐ洞窟です。
抗議活動家たちの白骨死体が Vault の扉の真ん前の地面に散乱しており、3つのプラカードが置かれています。

監督官のトンネル

このセクションは、監督官のオフィス（アトリウム/上層階にある）とメインエントランスを繋いでいます。

アトリウム/上層階

アトリウム

このアトリウムはメインエントランスと下層階を繋いでいます。
管理区とアトリウムの二つのサブセクションに分かれています。
アトリウムは Vault の上層階に位置しています。
この部屋へのアクセスは、食堂とアトリウムを繋ぐ階段で行われます。
「システム」が管理区とアトリウムを繋いでいます。
アトリウムの部屋は2つのフロア（上階と下階）に分かれています。
上階の部屋（ダイニング/マック家のリビング）は立ち入り禁止です。
アトリウムは Vault のメインルームであり、居住者による大規模な社交の集まりやスポーツに使用されます。
ウォーリーの誕生日パーティーの一つもここで行われ、彼と他の子供たちは「ミュータント狩り」をしました。

食堂

食堂の通知には、今後の野球の試合が告知されています。
管理区にある部屋は、監督官のオフィス、受付エリア（「Escape!」の最中にジョナス・パーマーがここで死んでいるのが見つかります）、オペレーション、セキュリティステーション、システム（「Escape!」の最中にフロイド・ルイスがここで死亡します）です。

下層階

これは Vault の最終セクションです。
「下層階」という名前ですが、実際には下層階とリアクターの二つのレベルをカバーしています。

教室
孤独な放浪者のアパート

下層階にある部屋は、クリニック、教室、ダイナー、デロリア家のアパート、トイレ、孤独な放浪者のアパートです。
リアクターレベルにある部屋は、発電機と貯蔵室（BBガンの射撃場）、フィルター室（浄水器）、リアクター室です。

メンテナンス部門

この部門は Vault の清掃と維持を担当しています。
この部門内の仕事には、ゴミ焼却担当、Pip-Boy プログラマー、廃棄物管理スペシャリスト、ジュークボックス技術者が含まれます。
メンテナンス技術者はユーティリティ・ジャンプスーツとワークブーツを着用しています。
技術者は時折、レンチや他の工具、そしてユーティリティ・キーを携帯しています。
この部門で働くことは、G.O.A.T.試験を受ける学生たちからは望ましくないものと考えられています。

居住者

アマタ、アルフォンス、ブッチ、スタンリー、ベアトリス、ジョナスなど多数のキャラクターが含まれます

注目の戦利品

医薬のボブルヘッド（クリニック内のジェームスのオフィスにあります）

ロック・イット・ランチャーの設計図と300キャップ （ジェームスのオフィス内、額縁に入った聖句の裏にある「中級」ロックの金庫の中にあります）

グロックナックバーバリアン（ダイナーでアマタから受け取ります。パーティー中に使用しなかった場合、Vault を去る前にタンスの中に置かれます）

ステルスボーイ（Vault 101 に戻った際、エアロック/除染コントロールステーションにあるスティーブ・アームストロングの死体から見つかります）

戦前のトースター（ダイナーのカウンターの裏にあります）

ブッチのつまようじ（「Trouble on the Homefront」の最中にブッチが所持しています）

改造ユーティリティ・ジャンプスーツ （「Trouble on the Homefront」を平和的に解決した際、アマタから与えられます）

注記

D.C.の廃墟にある Vault-Tec 本部では、Vault 101 の防爆扉のレプリカがロビーエリアに吊るされており、スポットライトで照らされています。
オープニングムービーでは、Vault の防爆扉の両側に2つの監視カメラが表示されます。
しかし、「Escape!」で Vault を去る際、防爆扉の上にはカメラが1つしか見えません。
「Trouble on the Homefront」では、カメラは消えています。
ゲーム開始時に Vault を出る際、Vault 101 のマップマーカーはマップ上でアクティブになりますが、Pip-Boy 3000 の「発見した場所」の統計には加算されません。
Vault 101 には4つのバージョンがあります。
1つ目は廊下とリビング、寝室だけで構成され、「Baby Steps」で使用されます。
2つ目は「Growing Up Fast」で孤独な放浪者が10歳になる誕生パーティーの時のものです。
3つ目は「Escape!」で見られるバージョンで、最後は「Trouble on the Homefront」で見られるわずかに変更されたバージョンです。
「Baby Steps」「Growing Up Fast」「Future Imperfect」で使用された Vault のバージョンはすべて「Escape!」で見られるマップに含まれており、プレイ可能エリアの外に配置されています。
また、ゲーム開始時にキャサリンがプレイヤーを出産するジェファソン記念館のセクションも同様です。
Vault 入口の外の道路上には「タイプB」のランダムエンカウンター発生場所があります。
高架道路を少し過ぎて南へ道路を辿ると発生させることができます。

Vault 入口からの風景

Vault 入口の外の見晴らしの良い場所は、仲間にした後の動物コンパニオン、ドッグミートの待機場所となります。
彼を解雇するとこの場所に向かいます。
また、この場所は、ザ・ピットへ続く列車のトンネル内にある奴隷の檻から解放された奴隷たちが、旅を生き延びた場合の目的地でもあります。 Fallout 76 のアップデート「Skyline Valley」の実写プロモーションビデオ内の地図に、この Vault が表示されました。この地図では、Vault 101 はメリーランド州、ワシントンD.C.の最北端付近、つまりポトマック川の北に位置しており、これは Fallout 3 のマップ位置とは異なります。

舞台裏

ダイナーには、初代 Fallout ゲームへのリファレンスがあります。
ジュークボックスの上の掲示板には、ダイナーでのビンゴ大会が宣伝されており、選ばれた数字は「13」で、「1等賞は1週間分の水配給！」となっています。
もう一つの Fallout へのリファレンスは、ブッチが自分のギャングの名前について尋ねられた際、なぜ「Vault Dweller（Vault 居住者）」なんて呼ばれたい奴がいるんだ？ と友人に問い返す場面です。
Vault 101 のいくつかの要素は、ジョージ・オーウェルの小説『1984年』にインスパイアされています。

これには Vault 番号（101号室）や監督官への献身（ビッグ・ブラザー）が含まれます。
Vault 内のあちこちで見られる「ハッピーメーカー」には、Fallout 4 の様々なポスターでも見られるメッセージが表示されています。
これらには「アメリカは生き続ける... お前の中で」「外の世界がお前を傷つけることは決してない」「安全第一」「清潔さは神性なり」「我々は皆一蓮托生だ」「重労働は幸せな仕事」「今日の薬はもう飲んだか？」といったメッセージが含まれます。
Vault 101 は、クロスプロモーションの「Fallout マッシュアップパック」のロケーションとして Minecraft にも登場します。

感想

実験の残酷さ: 「決して開かない Vault」という設定自体が、監督官の絶対的な権力を研究するための壮大な社会実験であったという事実は、Fallout シリーズらしい冷徹な設定ですね。

環境ストーリーテリング: 偽のラジオ放送や、外の世界への恐怖を植え付ける教育など、物理的な壁だけでなく精神的な壁によって居住者を閉じ込めていた初期の年月が、後の反乱へと繋がっていく流れが非常にドラマチックです。

現実世界とのリンク: ジョージ・オーウェルの『1984年』へのオマージュや、実在のファンへの献辞が含まれたメモなど、この場所が単なるゲームのステージを超えた意味を持っていることが再確認できました。
