サバイバルテント

DATE: 2025/12/22

Fallout 1st（有料メンバーシップ）加入者専用の、持ち運び可能な簡易拠点です。
通常のC.A.M.P.とは別に設置でき、無料のファストトラベル地点として利用できるのが最大の特徴です。

基本的な機能

どのタイプのテントにも、最低限以下の設備が備わっています。

  • スタッシュボックス

  • スクラップボックス

  • 寝袋またはベッド

  • クッキングステーション

種類による違い（重要）

アトムショップ等で入手できる「スキン（外見）」によって、設置されている設備が異なります。
特に作業台や弾薬ボックスの有無は利便性に大きく影響します。

1. 最高性能（機能全部入り） 最近登場した多機能なタイプです。
探索に最も便利です。

  • ビンテージキャンパー / ハウスボート

    • 特徴：細工師の作業台、弾薬保管ボックス、パンチカードマシン（SPECIAL変更）、楽器、ベッドが完備されています。

  • ガゼボ

    • 特徴：アーマー作業台がある珍しいタイプ。
      弾薬保管ボックス、楽器も完備。

  • エンクレイヴ・中継ステーション

    • 特徴：武器作業台があるタイプ。弾薬保管ボックス、ベッド付き。

2. 作業台付き（中機能） 基本的な機能に加え、解体や修理に便利な作業台が付いています。

  • APC（装甲兵員輸送車） / 放棄された小屋 / トラック

    • 特徴：細工師の作業台が付いています。

    • ※APCとトラックは寝袋ではなく「ベッド」扱い。

3. 弾薬ボックス付き

  • ヌカ・コーラのシート

    • 特徴：弾薬保管ボックスが付いていますが、楽器はありません。

4. 標準タイプ 見た目は異なりますが、機能はほぼ共通です。

  • 基本形 / レスポンダー / レイダー / 入植者 / ホリデー

    • 特徴：基本機能 ＋ 楽器（演奏してAP回復バフ付与）。

  • リーン・ツー

    • 特徴：基本機能のみ（楽器なし）。

使い方とルール

  • 設置方法 お気に入りホイール（PCならFキー/Tキー、CS機なら方向キー等）を開き、画面下に表示される「サバイバルテント」のボタンを押して設置します。

  • 移動（再設置） キャップ（通貨）を消費せず、何度でも無料で好きな場所に移動・再設置できます。

  • 設置場所の制限 平坦な地面である必要があります。既存の建物、他のプレイヤーのテントやC.A.M.P.と重なる場所には置けません。

  • 有効期限 Fallout 1stの期限が切れると、設置済みのテントも使用できなくなります（自動撤去、またはアクセス不可になります）。

感想

一言で言えば、「単なる便利機能を通り越して、ゲームプレイの快適性を劇的に変える『必須級』のシステム」という印象を強く受けます。

具体的に感じた点は以下の4点です。

1. スキンによる「性能格差」が驚くほど大きい

「見た目（スキン）によって機能が露骨に違う」という点です。

初期のテント：倉庫機能と料理のみ。

最新のテント（ハウスボートやビンテージキャンパー）：作業台、弾薬ボックス、パンチカードマシン（ステータス変更）まで完備。

単なる見た目の違いではなく、「どのスキンを持っているかで探索効率が段違いになる」という仕様は、後発のプレイヤーや特定の時期にプレイしていなかった人にとっては、少し歯痒い部分かもしれません。逆に言えば、「作業台付きテント」がアトムショップに並んだ時は、最優先で確保すべきだということがよく分かります。

2. 「重量オーバー」のストレスを消滅させるシステム

Falloutシリーズ恒例の悩みである「重量」を、現地で即座に解決できるのは革命的です。

特に「スクラップボックス（ジャンク無限収納）」と「作業台（武器防具を解体してジャンク化）」がセットになったテント（APCやエンクレイヴ等）があれば、どこでも身軽になれるため、拠点に戻る手間が省けます。
これを知ってしまうと、もうテント無しの生活には戻れないほどの依存性があるシステムだと感じます。

3. 「第2のファストトラベル地点」としての戦略性

収納機能だけでなく、「無料で移動できるポイントを任意に作れる」という点が非常に強力です。

スコーチビースト・クイーン戦の付近。

よく行く素材集め場所。

駅やベンダーから遠いエリア。

これらにポンと置いておくだけで、キャップの節約と時間の短縮になります。
C.A.M.P.を動かすとコストがかかりますが、テントはタダで動かせるため、「機動力の要」と言えます。

4. Fallout 1st加入の最大の動機づけ

このサバイバルテント（特にスクラップボックス機能との連携）こそが、月額サービス「Fallout 1st」に加入する最大のメリット（あるいは「これがないと不便すぎる」という誘導）になっていると改めて感じました。

サバイバルテントは、単なる「ポータブルな家」ではなく、「インベントリと自由な移動地点を提供する最強のツール」です。
特に、「細工師の作業台」や「弾薬保管ボックス」付きのスキンを持っているかどうかで、アパラチアでのQoLが天と地ほど変わると言っても過言ではありません。

私はビンテージキャンパーのサバイバルテント使ってます。
アパラチアラジオも付いてるのがGood!
